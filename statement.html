The place where the silence is so loud that I can no longer hear the voice inside my head.
Where there is nowhere to go, but I can go anywhere.
Where everything ends, and begins.

In this project, I made a space-inspired emulation. The site is built using HTML/CSS, while the interactivity and animation is created using the p5.js library.
